# hotel-fb-cost-calculator
 I built an F&B Cost Impact Calculator for Hotel Operations.
As a Hotel Management student specializing in F&B, I wanted to go beyond theory — so I designed a full financial decision tool that helps hotel managers and F&B executives understand how food and beverage costs affect profitability.
What it does:

→ Calculates Food Cost %, Beverage Cost %, Labor Cost impact in real time

→ Industry benchmark comparison (Excellent / Good / Warning)

→ Scenario Analysis with live sliders

→ What-If Simulator (Reduce Food Cost by 1%, 3%, increase revenue, etc.)

→ Per Cover Analysis (Avg Check, Cost per Cover, Profit per Cover)

→ Generates a Management Report with recommended actions
Why I built it:

During my internship at ITC Grand Bharat (A Luxury Collection Retreat), I worked in banquet operations and saw how tightly F&B cost percentages are monitored. A 1% reduction in food cost at a ₹50L/month operation saves ₹6 lakh annually. That context motivated this project.
